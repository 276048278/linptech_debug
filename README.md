# 领普科技部署调试软件

> 20180921
> - 用于领普科技产品部署安装调试
> - python开发，支持各平台
> - 需要与领普调试器配合使用

## 1 软件功能

- 领普接收器与开关(限433协议）ID的读取
- 控制接收器通断，模拟开关发送指令
- 接收器与开关单个批量配对及配对清除
- 配对完成后，发射器与接收器信号强度调试
- 配对完成后接收器通断自动测试
  - 顺序通断测试
  - 模拟开关信号控制测试
  - 发射器快速通断测试

## 2 使用

### 2.1 运行环境准备

- 电脑安装python3环境
- pip安装必须的python库 `pip install -r requirements`
- 电脑连接领普调试器
- python3运行main.py `python3 main.py`
